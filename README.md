# BDBWin 5.0 — Landing Page

Marketing landing page for **BDBWin 5.0**, the complete rebuild of the BDBWin ERP platform by [Megacom GmbH](https://megacom.de).

🔗 **Live site:** https://ahmmedtowfique.github.io/bdbwin-landing/

---

## What is BDBWin 5.0?

BDBWin 5.0 is a full ERP rebuild — orders, finance, warehouse, procurement, POS — with a live intelligence layer and an AI decision layer that gives business owners a plain-language morning briefing in 60 seconds.

This repo is the **landing page** for the product, built to collect early access signups.

---

## Tech stack

- HTML / CSS / Vanilla JS
- Hosted via GitHub Pages

---

## Run locally

```bash
git clone https://github.com/ahmmedtowfique/bdbwin-landing.git
cd bdbwin-landing
# Open index.html in your browser — no build step needed
```

---

## Deploy

Pushes to `main` deploy automatically via GitHub Pages.

To configure: **Settings → Pages → Branch: main / root**
